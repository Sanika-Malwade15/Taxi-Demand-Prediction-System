
# 🚖 Taxi Demand Prediction

###  Predicting taxi demand using Machine Learning

**AI-powered system that forecasts taxi demand based on historical trip data, time patterns, and location features to optimize ride availability.**

##  Features

*  Predicts taxi demand for different time intervals
*  Time-based feature engineering (hour, day, peak hours)
*  Location-based analysis
*  Data visualization for demand trends
*  Machine learning model for prediction



##  Dataset

🔗 **Dataset Link:**
(https://www.kaggle.com/datasets/elemento/nyc-yellow-taxi-trip-data)



## ⚙️ How It Works

1. Data preprocessing and cleaning
2. Feature engineering (time & demand patterns)
3. Exploratory Data Analysis (EDA)
4. Model training using ML algorithms
5. Demand prediction

---

##  Sample Input Features

* Hour of the day
* Day of the week
* Location ID
* Peak hour indicator

---

##  Model Used

* XGBoost

---

##  Output

* Predicted demand values
* Visual graphs showing demand trends
